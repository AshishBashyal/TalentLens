# Kaggle Jobs Dataset

Place the downloaded Kaggle CSV files in this folder only for local development.

Expected files:

- `postings.csv`
- `companies.csv`
- `company_specialities.csv`
- `employee_counts.csv`
- `skills.csv`
- `salaries.csv`
- `job_skills.csv`
- `job_industries.csv`
- `industries.csv`
- `benefits.csv`

The raw CSV files are intentionally ignored by Git because they are large and may have license restrictions.

Before using this dataset in public reports or notebooks, confirm the license on the Kaggle dataset page.
